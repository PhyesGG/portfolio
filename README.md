## Technologies utilisees

- Next.js 14
- TypeScript
- TailwindCSS
- Framer Motion

## Installation

```bash
# Cloner le repository
git clone https://github.com/PhyesGG/portfolio.git

# Installer les dependances
npm install

# Lancer en mode developpement
npm run dev
```

Le site sera accessible sur `http://localhost:3000`
